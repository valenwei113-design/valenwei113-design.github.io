# Valen Wei

Personal website — a visual journal of travel, life, and learning.

## Pages

- **Home** (`homepage.html`) — Full-screen video slideshow cycling through Iceland, Norway, diving trips, and more. Click the name to enter the Journey tree.
- **Journey** (`journey.html`) — An interactive SVG tree with navigation buttons to explore:
  - Vibe Coding — web application projects
  - 德国留学 — study abroad in Germany
  - 北欧游学 — Nordic adventure
  - 打工换宿 — work exchange, New Zealand & Norway
  - 旅居日历 — interactive travel calendar
  - Xiaohongshu — social cross-post highlights
- **旅居日历** (`journey-calendar.html`) — Morandi-colored travel log with searchable city database, year/month grid, and localStorage persistence.

## Interactions

- Home page video slideshow with music player and spinning vinyl animation
- Journey tree with falling leaves (CSS + SVG, ~2300 static leaves + 22 animated)
- Photo lightbox — click any gallery image to zoom in
- Calendar with city picker, custom country entries, and year summary stats
- Card hover lift effects and fade-up entry animations

## Tech

Plain HTML + Tailwind CSS + D3.js (maps) + SVG animations. No frameworks, no build step. Hosted on GitHub Pages.
