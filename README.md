# Valen Wei

Personal homepage — a visual journal of travel, life, and learning.

## What's here

- **Home** — Full-screen video slideshow cycling through Iceland, Norway, diving trips, and more. Ambient background music with a turntable toggle. Click the name to enter the Journey tree.
- **Journey tree** — An interactive SVG tree with glowing firefly buttons. Hover to reveal chapter labels, click to explore four chapters:
  - **Norway** — A road trip through the fjords
  - **Germany** — Study abroad guide: university types, visa costs, APS process
  - **Career** — Reflections across state-owned and foreign enterprises
  - **Living** — Photo walls from life across different countries
- **Vlog** — Curated travel clips and daily moments.
- **Xiaohongshu** — Social cross-post highlights.

## Interactions

- Video slideshow auto-advances on the home page
- Music player with spinning vinyl animation, auto-plays on load
- Firefly hover glow + label reveal on the Journey tree
- Falling leaves animation (CSS + SVG, ~2300 static leaves + 22 animated)
- Photo lightbox — click any gallery image to zoom in
- Card hover lift effects on Germany and Career pages
- Corner navigation with hover fade-in

## Tech

Plain HTML + Tailwind CSS + SVG animations. No frameworks, no build step. Hosted on GitHub Pages.
